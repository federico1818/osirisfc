# Osirisfc

## First

Set script ```run.sh```  executable as program

```sh
chmod u+x run.sh
```

## Docker

Run

```sh
docker compose -f ./docker/docker-compose.yml up --detach
```

Down

```sh
docker compose -f ./docker/docker-compose.yml down
```
